# 🐹 FrienDiary (햄식이)

> **"오늘 하루는 어땠어? 내가 들어줄게!"**
> 
> AI 친구 **햄식이**와 편하게 대화하고, 당신의 하루를 자동으로 일기로 기록해 보세요.

<br>

## 프로젝트 구경해보기

[데모 링크]https://milkeon.github.io/talkDiary/

## 📌 프로젝트 소개

**FrienDiary**는 AI 캐릭터 '햄식이'와의 자연스러운 대화를 통해 하루를 돌아보고, 그 내용을 자동으로 일기로 변환해주는 감성 다이어리 웹 앱입니다.

글쓰기가 부담스러운 날에도, 그냥 친구에게 말을 걸듯 오늘 있었던 일을 이야기하면 햄식이가 알아서 예쁜 일기로 정리해줍니다. 🌟

<br>

## ✨ 주요 기능

### 💬 1. 친구 '햄식이'와의 대화
- AI 친구 햄식이와 도란도란 이야기를 나눕니다.
- 사용자의 이름을 불러주며, 감정에 맞는 따뜻한 리액션을 건넵니다.
- `[오늘도 수고했어~]` 태그와 함께 작별 인사를 하면 자동으로 일기 작성 모드로 전환됩니다.

### 📝 2. 자동 일기 생성
- 대화가 끝나면 햄식이가 대화 내용을 바탕으로 **나의 목소리**로 오늘의 일기를 요약해줍니다.
- 대화 분량에 비례해 정직하고 성숙한 문체로 일기를 작성합니다.

### 🎙️ 3. 음성 대화 지원 (STT / TTS)
- **마이크 아이콘**을 눌러 말로 대화할 수 있습니다.
- 2초간 침묵이 감지되면 자동으로 메시지가 전송됩니다.
- 햄식이의 목소리를 직접 들을 수 있으며, 말하는 속도도 조절 가능합니다.

### 📅 4. 기록 보관 및 요약
- **지난 기록**: 달력 방식으로 과거 일기를 다시 읽어볼 수 있습니다.
- **주간 요약**: 최근 3일간의 일기를 분석해 한 주의 흐름을 요약합니다.
- **감정 통계**: 이번 주의 주요 기분과 키워드를 시각화해 보여줍니다.

### 🎨 5. 감성 UI/UX
- 은하수가 쏟아지는 듯한 **Stardust 애니메이션** 효과를 제공합니다.
- 데스크톱과 모바일(하단 바 형식) 모두 최적화된 반응형 디자인을 지원합니다.
- 입력창에 커서가 항상 포커스되어 있어, 키보드로 즉시 대화를 시작할 수 있습니다.

## 🛠 기술 스택

| 영역 | 기술 |
|------|------|
| **Frontend** | HTML5, CSS3 (Vanilla), JavaScript, jQuery v3.7.1 |
| **Database** | Firebase Firestore |
| **AI Engine** | Google Gemini API (`gemma-3-27b-it`) |
| **Voice** | Web Speech API (Recognition & Synthesis) |

<br>

## 🎨 디자인 포인트

- **햄식이 아바타**: 모바일에서도 언제든 말을 걸 수 있도록, 왼쪽 하단 아바타 옆에 마이크 버튼을 배치했습니다.
- **로고 클릭**: 상단 로고(`FrienDiary`)를 클릭하면 언제든지 홈(기록하기) 화면으로 돌아옵니다.
- **빠른 입력**: 엔터 키 하나로 간편하게 메시지를 전송할 수 있습니다.
