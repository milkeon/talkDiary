# 🐹 햄식이 (FrienDiary)

> **"오늘 하루는 어땠어? 내가 들어줄게!"**  
> 세상에서 가장 다정한 친구 **햄식이**와 대화하고, 당신의 소중한 오늘을 일기로 기록해 보세요.

---

## ✨ 핵심 기능

### 1. 다정한 친구 '햄식이'와의 대화
- 친구 **햄식이**와 도란도란 이야기를 나눕니다.
- 사용자의 이름을 불러주고, 세심하게 감정 리액션을 해주는 친구입니다.
- **[오늘도 수고했어~]** 태그와 함께 따뜻한 작별 인사를 건넨 뒤 일기를 작성합니다.

### 2. 마법 같은 자동 일기 생성
- 대화가 끝나면 햄식이가 대화 내용을 바탕으로 오늘 하루를 **나의 목소리**로 요약해 줍니다.
- 대화 길이에 비례하여 정직하고 성숙한 문체로 일기를 작성합니다.

### 3. 음성 대화 지원 (STT / TTS)
- **마이크 아이콘**을 눌러 말로 대화할 수 있습니다. (2초 침묵 시 자동 전송)
- 햄식이의 목소리를 직접 들을 수 있으며, 말하는 속도 조절도 가능합니다.

### 4. 한눈에 보는 기록과 요약
- **지난 기록**: 달력 방식으로 과거의 일기를 다시 읽어볼 수 있습니다.
- **주간 요약**: 최근 3일간의 일기를 분석해 주간 흐름을 요약해 줍니다.
- **감정 통계**: 이번 주의 주요 기분과 키워드를 분석해 시각화합니다.

### 5. 세련된 감성 UI/UX
- 은하수가 쏟아지는 듯한 **Stardust 애니메이션** 효과를 제공합니다.
- 데스크톱과 모바일(하단 바 형식) 모두에 최적화된 반응형 디자인을 지원합니다.

---

## 🛠 기술 스택

- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (jQuery v3.7.1)
- **Backend / DB**: Firebase Firestore Integration
- **AI Engine**: Google Gemini API (`gemma-3-27b-it` 모델 활용)
- **Voice**: Web Speech API (Recognition & Synthesis)

## 🎨 디자인 포인트

- **햄식이의 위치**: 모바일에서도 언제든 말을 걸 수 있도록 왼쪽 하단 아바타 옆에 마이크 버튼을 배치했습니다.
- **로고**: 상단 로고(`FrienDiary`)를 누르면 언제든지 홈(기록하기) 화면으로 돌아옵니다.
- **인터랙션**: 입력창에 커서가 항상 잡혀 있어 키보드로 즉시 대화가 가능하며, 엔터 키로 간편하게 전송할 수 있습니다.
